# ![ logo ](https://github.com/wladislawmakowski/passt/blob/master/Logo.png) &ensp; passt
> passt â€” is a simple utility for KeePassXC purposed on keeping the password of the KeePass database in safety

### Options:

| Option | Description                                        |
|--------|----------------------------------------------------|
| -init  | Set up a new password                              |
| -get   | Copy password to the clipboard and start KeePassXC |
| -u     | Update the key                                     |
| -new   | Set up a new password                              |

### Installation:
#### * Debian (Ubuntu, Mint, etc.)
```bash
$ git clone https://github.com/wladislawmakowski/passt.git
$ cd passt
$ sudo ./setup.sh
```
